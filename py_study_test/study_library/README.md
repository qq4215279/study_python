    Jupyter 不仅能够对 Python 代码进行展示和格式化，还能够保存用户的历史代码和结果以及数据分析结果。这些结果可以在后期随时查看和修改，使得 Python 的学习和应用变得更加方便和高效。
    安装：pip install jupyter
    启动：jupyter notebook 或 jupyter lab
    打开浏览器，输入 http://localhost:8888 即可访问 Jupyter 主页。
